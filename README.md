# Next.js Server Actions Demo

This is a mini learning project built while exploring **React Server Components (RSC)** and **Server Actions** in Next.js.

## Features
- React Server Components (RSC)
- Server Actions for form submission
- Saving form data to a dummy JSON file
- Async data fetching with `async/await`
- Suspense for loading states
- Error handling with Error Boundaries

## Tech Stack
- Next.js (App Router)
- React
- Server Actions
- Node.js (fs/promises)

## Purpose
This project is created for learning and demonstration purposes to understand how modern Next.js handles server-side logic without traditional APIs.
